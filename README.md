# michaelawunikofi

Personal portfolio site for Michael Kofi Awuni — built with Rust and compiled to WebAssembly.

Live at: **https://michael-cmd-sys.github.io/michaelawunikofi/**

## Tech Stack

- **[Dioxus](https://dioxuslabs.com/) 0.7** — Rust UI framework targeting WebAssembly
- **Tailwind CSS 4** — utility-first styling with the `tailwind-animations` plugin
- **wasm-bindgen / web-sys / js-sys** — browser API bindings (IntersectionObserver for scroll animations)
- **GitHub Pages** — static hosting, deployed from the `/docs` directory on `main`

## Project Structure

```
├── src/
│   └── main.rs          # All components (App, Navbar, Hero, About, Projects, TechStack, Contact, Footer)
├── assets/
│   └── tailwind.css     # Compiled Tailwind output (generated by dx)
├── docs/                # Built release output — committed and served by GitHub Pages
├── tailwind.css         # Tailwind source / theme definitions
├── Dioxus.toml          # Dioxus config (base_path, tailwind paths, out_dir)
└── Cargo.toml
```

## Development

Requires the [Dioxus CLI](https://dioxuslabs.com/learn/0.7/getting_started):

```bash
cargo install dioxus-cli
```

Start the dev server with hot-reload:

```bash
dx serve
```

Runs on `http://localhost:8080` by default.

## Deploying to GitHub Pages

The site is deployed by building locally and committing the output to `docs/`. GitHub Pages is configured to serve from the `/docs` folder on the `main` branch — no CI build step required.

Build and update `docs/`:

```bash
dx build --release
cp -r target/dx/michaelawunikofi/release/web/public/{index.html,assets} docs/
git add docs/
git commit -m "deploy: update build"
git push
```

GitHub Pages picks up the change within ~1 minute.
